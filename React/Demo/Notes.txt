please make sure NodeJS is install in you system in order to run React application.
clone this React app from: github.com
then goto cloned add directory
then run this command: 
    ~ "npm install"
    ~ this will take few minutes may be 5 minutes to install all dependencies.
once installation complete
then run this command to start React app:
    ~ "npm start"
it will take may be 1-3 to start your application
    ~ by default app will run on: http://localhost:3000
